- **Background:** `#111827` / `#1f2937`

- **Borders:** `#374151`
- **Accent colors:** Same as light theme

## Animations

### CitationsList

- **Hover:** Border color change + subtle shadow
- **Duration:** 0.2s ease

### PersonalityIndicator

- **Expand/Collapse:** Slide down animation
- **Duration:** 0.2s ease-out
- **Button Active:** Scale(0.98)

### RAGVisualizer

- **Slide In:** Opacity + translateY
- **Icon Bounce:** Continuous bounce animation
- **Pulse:** Expanding circle animation
- **Progress Bar:** Width transition 0.5s ease-out
